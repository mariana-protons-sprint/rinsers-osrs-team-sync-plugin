# Team Sync RuneLite Plugin

Private RuneLite plugin for syncing live account state to the OSRS Team Sync backend.

## Current state

This plugin project is an internal starter intended for team deployment and testing.

## Required config

- Backend URL
- Team ID
- Client ID
- Profile Label
- Token
- Send inventory toggle
- Send equipment toggle

## Intended use

- first internal tester installs and validates live sync
- then expand to additional team accounts
- later decide whether to keep private or adapt to Plugin Hub standards

## Metadata

Main class: `com.richard.teamsync.TeamSyncPlugin`
Metadata file: `src/main/resources/runelite-plugin.properties`
